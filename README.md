# Dogs

The project is a web application developed in Angular, inspired by the design of the Dogs project from Origamid. It is a social network for pets where users can create accounts, share photos of their pets, interact with others through likes and comments, and engage with the pet-loving community.

#
Developed using [Angular CLI](https://github.com/angular/angular-cli) version 19.0.2.

#
To start a local development server, run:

```bash
ng serve
```

#
To build the project run:

```bash
ng build
```
